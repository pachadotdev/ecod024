Replication of ‘Optimal Fiscal Policy in a Business Cycle Model’ by Chari, Christiano, and Kehoe (1994)

The goal of cck94 is to replicate the results from Chari et al. (1994) using time iteration, the Endogenous Grid Method (ECM) by Maliar and Maliar (2013), and projections (original method).

For the time iteration approach I used a Monte Carlo 1st-order Linear Quadratic (LQ) approximation and a 2nd-order perturbation following Benigno and Woodford (2005).

I do not have a MATLAB license, so I looked at Grohe-Uribe solvers to adapt a bit an write a small program using Armadillo.

The files dev/replicate-cck94-table4.R and dev/replicate-cck94-table5.R contain the code to compute the tables below. dev/table4.r and dev/table5.r contain the code to organize the original results.

Installation

You can install the development version of cck94 from GitHub with:

# install.packages("pak")
pak::pak("pachadotdev/ecod024/cck94")
